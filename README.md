# superinstance-live

**DAW session controller** — manages live audio/DAW sessions across the SuperInstance ecosystem. Synchronize musical state, control playback, and coordinate real-time audio processing.

## What This Gives You

- **Session management** — create, join, and synchronize DAW sessions
- **Transport control** — play, stop, seek, and tempo changes across connected clients
- **State sync** — real-time synchronization of musical parameters via CRDTs
- **Audio routing** — coordinate audio processing pipelines across agents

## How It Fits

Live performance layer for the `symplectic-music` and `counterpoint-engine` ecosystem. Uses `SmartCRDT` for state sync and `conservation-spectral` for real-time spectral processing.
